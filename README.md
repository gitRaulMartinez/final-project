# PROYECTO FINAL

Proyecto final del curso **Digitalers Node.js Developer**

## Comandos

Iniciar mongo

```bash
    mongod
```

Iniciar proyecto en desarrollo

```bash
    npm run dev
```

Estilos CSS tailwind

```bash
    npx tailwindcss -i ./tailwind/input.css -o ./public/css/tailwind.css --watch
```
